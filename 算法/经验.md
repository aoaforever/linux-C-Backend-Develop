4. 找出一个数中二进制1的个数：
```cpp
int count = 0;
while(N!=0)
{
    int rightone = N & (~N + 1);
    count++;
    N ^= rightone; //每次抹掉最右侧的1. 这么做避免了循环32位int。
}
return count;
```
